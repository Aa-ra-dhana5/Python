# OOP is object oreinted programming 
-is programming language model organized around objects rather than action(function) and data rather than logic


1. Encapsulation: Bundling data and methods, restricting direct access to some attributes.

2. Inheritance: Reusing and extending existing classes.

3. Polymorphism: Same interface, different implementations.
          1.Compile time polymorphism  - method overlaoding
          2.Runtime polymorphism -- method overriding , operator overloading , Duck typing

4. Abstraction: Hiding internal details, exposing only necessary functionality.


--> Class -group of f attributes and methods
attributes - variable that contains data
method - perform action
